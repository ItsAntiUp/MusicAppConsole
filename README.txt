---------------------------------------------------------
"save.bin" - iš anksto paruoštas dainų sąrašas testavimo tikslams.
---------------------------------------------------------
Taip pat aplanke "SONGS" pridedu keletą dainų, kurios skirtos išsibandyti funkciją: Load from path.
Ši funkcija užkrauna atitinkamus failus į sistemą iš jūsų nurodyto katalogo.
Užtenka parašyti ".", kad failai būtų užkraunami iš einamojo katalogo.
Norint pasiekti pridėtą aplanką reikėtų rašyti "SONGS", jei paleidžiate .exe failą arba ".\SONGS", jei paleidžiate per kompiliatorių.
